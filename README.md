## Hyper Spectra
A rhythm game for the Nintendo 3DS.

## Video Conversion
`ffmpeg -i "InputVideo.videoFileExtensionHere" -vcodec theora -b:v 2500k -q:v 7 -g 30 -pix_fmt yuv420p -vf scale=400:-1 -r 24 -an "Output.ogv"`
